# Human–AI Interaction Study

This repository contains the web-based experimental implementation for an MSc thesis project investigating human interaction with AI systems during logical reasoning tasks.

## Overview

The study examines how individuals revise their decisions and confidence levels after receiving advice from an AI system of varying reliability. The experiment is implemented using jsPsych and includes:

- Conditional reasoning and syllogism tasks
- AI advice manipulation (varying reliability across phases)
- Confidence ratings (pre- and post-AI advice)
- Behavioral measures of decision revision

## Study Design

Participants complete 45 reasoning tasks divided into three phases.  
For each task:

1. Initial answer  
2. Initial confidence rating  
3. AI recommendation  
4. Final answer  
5. Final confidence rating  

Encouragement screens are included to reduce fatigue.

## Data Collection

The experiment is hosted online and stores anonymized data via a secure Google Sheets endpoint. No personally identifying information is collected.

## Data Ethics Statement

This study collects only anonymized behavioral data related to task performance and confidence ratings.

No personally identifying information (e.g., names, emails, IP addresses) is collected or stored.

Participants are informed that:
- Participation is voluntary
- They may withdraw at any time by closing the browser
- Data are stored under an anonymous participant code
- Data are used solely for academic research purposes

All data handling complies with GDPR principles of data minimization, anonymity, and research use.
